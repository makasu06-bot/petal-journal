yjff Install Guide

Run each step in your Mac Terminal in order

Step 1
Install Playwright One-time
pip3 install playwright && python3 -m playwright install chromium

Only needed the first time. Takes ~1 minute.

Step 2
Download the new script
curl -s https://raw.githubusercontent.com/makasu06-bot/petal-journal/main/yjff.py -o /tmp/yjff_new.py
Step 3
Test it (your installed yjff stays untouched)
python3 /tmp/yjff_new.py https://justfor.fans/TBOCgoonerXham --limit 1

If you see a video download, it's working. Confirm it has audio before Step 4.

Step 4
Replace your installed script
cp /tmp/yjff_new.py /Users/marcus/bin/yjff

Only run this after Step 3 confirms it works.
